🔄 DEPLOYMENT OPTIONS
═══════════════════════════════════════════════════════════════════════════════

OPTION 1: GitHub Pages (Free)
  1. Create GitHub account
  2. Upload files to repository
  3. Enable Pages in settings
  4. Access at: yourusername.github.io/repo

OPTION 2: Netlify (Free, Easiest)
  1. Visit: netlify.com
  2. Drag & drop your folder
  3. Get live URL instantly
  4. Done!

OPTION 3: Vercel (Free, Fast)
  1. Visit: vercel.com
  2. Import from GitHub
  3. Auto-deploys on updates
  4. Done!

OPTION 4: Any Web Host
  1. FTP upload files to server
  2. Make sure index.html is in root
  3. Access via your domain
  4. Done!

OPTION 5: Local Server
  Windows:
    python -m http.server 8000
    
  Mac/Linux:
    python3 -m http.server 8000
    
  Node.js:
    npx http-server


💡 CUSTOMIZATION TIPS
═══════════════════════════════════════════════════════════════════════════════

CHANGE COLORS
Open style.css and edit:
  --color-primary: #00d4ff;      (main cyan)
  --color-dark: #000000;         (text)
  --color-secondary: #ffffff;    (white)

CHANGE LOGO
Replace ctex-logo.webp with your logo:
  Make sure it's named ctex-logo.webp
  Or update the filename in index.html

CHANGE VIDEO
Replace ctex-logo-2.mp4 with your video:
  Make sure it's named ctex-logo-2.mp4
  Or update the filename in welcome.html

CHANGE TEXT

  PDPR, How to Use → Your nav links
