## What's New in v1.2.0

### 1️⃣ Mobile Navigation (Hamburger Menu)
- **Problem**: Navigation out of bounds on mobile
- **Solution**: Responsive hamburger menu on mobile (≤768px)
- **Files**: `index.html`, `style.css`, `script.js`
- **Features**: Smooth animations, auto-close, click-away detection

### 2️⃣ PDPR Privacy Policy
- **Problem**: No formal privacy documentation
- **Solution**: Complete 2,000-line PDPR-compliant policy
- **Files**: `PDPR_PRIVACY_POLICY.md`, updated `index.html`
- **Coverage**: GDPR compliant, CCPA compliant, 3-phase cookie strategy

### 3️⃣ Canvas Optimization
- **Problem**: Basic canvas rendering, not optimized
- **Solution**: Smooth curves, better centering, improved rendering
- **Files**: `style.css`, `script.js`
- **Features**: Quadratic Bezier curves, device pixel ratio aware, 60 FPS

### 4️⃣ How-to-Use Guide
- **Problem**: No user documentation
- **Solution**: Interactive HTML guide with tutorials and FAQ
- **Files**: `how-to-use.html` (2,500 lines)
- **Sections**: Standard guide (5 steps), AI guide (7 steps), comparison, tips, FAQ
