### 1. API Server
- **Technology:** CloudFlare Workers + Wrangler
- **Port:** 8000
- **Status:** Auto-started and monitored
- **Auto-restart:** Yes (after 5 failed checks)
- **Endpoints:**
  - GET /api/v1/health
  - POST /api/v1/signature/analyze
  - POST /api/v1/signature/sign
  - GET /api/v1/version

### 2. Health Monitor
- **Check Interval:** 30 seconds
- **Action:** Logs health status
- **Failure Handling:** Auto-restart after 5 failures
- **Logging:** Complete audit trail

  - API requests

- **Format:** CSV files
- **Update Frequency:** Every 30 seconds
- **Storage:** ai-data/AI_Reports_YYYY-MM/

- **Frequency:** Auto-generated at month-end
- **Contents:**
  - Processing statistics
  - AI accuracy metrics (tracked improvements)
  - User behavior insights
